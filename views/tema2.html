Cloud Filestore

es un servicio de almacenamiento de archivos gestionado para aplicaciones que requieren una interfaz de sistema de archivos y un sistema de archivos compartido para los datos. El monitoreo de Google Cloud Platform (GCP) Filestore le permite garantizar la salud y la disponibilidad de las instancias, solucionar los errores rápidamente y ofrecer un alto rendimiento.
